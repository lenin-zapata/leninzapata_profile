Resumen del Proyecto

Este proyecto es un flujo de trabajo de automatización de procesos robóticos (RPA) desarrollado en n8n. Su objetivo es optimizar la validación de datos integrando la toma de decisiones humana en procesos automatizados. El robot extrae datos vía API, solicita validación por correo interactivo y actualiza bases de datos automáticamente según la respuesta.

El Problema a Resolver

En los negocios, los procesos de validación manual suelen convertirse en cuellos de botella que pierden tiempo y dinero. La automatización pura a veces falla sin supervisión. Esta solución elimina el problema combinando la velocidad del robot con el criterio humano.

El Robot: Flujo de Trabajo

  • Extracción: Conexión a API (JSONPlaceholder) para obtener tareas recientes.
  • Validación Humana: Envío de email interactivo al gestor para aprobar/rechazar con un clic.
  • Actualización: El robot actualiza Google Sheets automáticamente basado en la decisión.
